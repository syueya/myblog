ssh连接

首先去绿联开启ssh：点击控制面板-终端机，打开启用ssh的开关，端口一般默认是22。记得开启/关闭操作后点击应用按钮使更改生效~

# ssh连接方法

可以使用系统自带的终端用命令行连接，也可以用专门的客户端来连接。

# 命令行连接

语法：ssh [options] [user@]hostname [command]

参数说明：

  • [user@]hostname
    • user : 要连接的远程主机的用户名
    • hostname 远程主机的名称或IP地址。
  • [command]：可选的在远程主机上执行的命令。

常用选项：

  • -l <user>：指定要登录的用户。
  • -p <port>：指定连接到远程主机的端口号，默认是22。
  • -i <identity_file>：指定身份验证文件（私钥文件）。
  • -v/-vv/-vvv：调试模式（Verbose），输出连接细节（排错用：-v 基础，-vvv 最详细）

示例：

# 如果你的用户名是xia，绿联IP地址是192.168.31.15
ssh xia@192.168.31.15
ssh -l xia 192.168.31.15

# 如果指定了ssh端口为9922
ssh -p 9922 xia@192.168.31.15

# 指定私钥文件
ssh -i ~/.ssh/my_key xia@192.168.31.15

# 客户端连接

安装一个你喜欢的客户端，比如mobaxterm，我以Termius为例:点击new host后填写主机的地址和端口，以及登录的用户名与密码，然后点击connect连接。
