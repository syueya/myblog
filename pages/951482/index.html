# 安装镜像

1、服务器安装docker镜像：

# 镜像：razeencheng/wxapiproxy

docker run -d --name wxapiproxy --restart unless-stopped -p 9080:80 razeencheng/wxapiproxy
1

compose文件：

services:
  wxapiproxy:
    image: razeencheng/wxapiproxy
    container_name: wxapiproxy
    restart: unless-stopped
    ports:
      - 9080:80

# 镜像：nicholastzw/wxchat

docker run -d --name wxchat --restart=always -p 9080:80 nicholastzw/wxchat:latest
1
  • 说明：9080可以随意一个未被占用的端口

2、开放9080端口

3、以razeencheng/wxapiproxy为例：访问服务器的IP:9080，页面出现则表示成功

Welcome to Weixin API Proxy Page!!
1

# 使用

1、到创建的应用中，找到设置企业可信IP, 点击配置，填入我们的服务器IP。

2、在消息推送代理 中填写我们搭建的代理服务的地址，即 http://服务器IP:9080
