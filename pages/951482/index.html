# 安装镜像

1、服务器安装docker镜像：

  • 镜像：razeencheng/wxapiproxy
docker run -d --name "wxapiproxy" \
	 -p "9080:80" \
	 --restart unless-stopped \
	 razeencheng/wxapiproxy
1
2
3
4
  • 或者镜像：nicholastzw/wxchat
docker run --name wxchat --restart=always -d -p 9080:80 nicholastzw/wxchat:latest
1
  • 说明：9080可以随意一个未被占用的端口

2、开放9080端口

3、以razeencheng/wxapiproxy为例：访问服务器的IP:9080，页面出现则表示成功

Welcome to Weixin API Proxy Page!!
1

# 使用

1、到创建的应用中，找到设置企业可信IP, 点击配置，填入我们的服务器IP。

2、在消息推送代理 中填写我们搭建的代理服务的地址，即 http://服务器IP:9080
