首先，克隆仓库：

```bash
git clone https://github.com/gethomepage/homepage.git
```
然后安装依赖并构建生产包（这里我使用pnpm，您可以根据需要使用npm或yarn）：

```bash
pnpm install
pnpm build
```
如果这是您第一次启动，请将 `src/skeleton`目录复制到`config/`以填充初始示例配置文件。

最后，运行服务器：

```bash
pnpm start
```
