官网：https://github.com/Dooy/chatgpt-web-midjourney-proxy

chatgpt-web-midjourney-proxy是一款项目集合了chatgpt web、midjourney、gpts、suno、luma、runway、viggle、flux、ideogram、realtime、pika、udio。


## 1、chatgpt-web-midjourney-proxy部署

```
services:
  chatgpt-web-midjourney-proxy:
    image: ydlhero/chatgpt-web-midjourney-proxy:latest
    container_name: chatgpt-web-midjourney-proxy
    restart: always
    ports:
      - 6015:3002
    environment:
      - OPENAI_API_KEY=sk-xxxxx
      - OPENAI_API_BASE_URL=https://api.openai.com
      - MJ_SERVER=https://your-mj-server:6013
      - MJ_API_SECRET=your-mj-api-secret
      - LUMA_SERVER=https://your-luma-server:8000
      - LUMA_KEY=your-luma-key
      - SUNO_SERVER=https://your-suno-server:8000
      - SUNO_KEY=you-suno-key
```


## midjourney-proxy API docker部署

如果有需要midjourney的话还得部署这个。

```
services:
  chatgpt-web-midjourney-proxy:
    image: novicezk/midjourney-proxy:2.5.5
    container_name: mj6013
    restart: always
    ports:
      - 6013:8080
    environment:
      - mj.discord.guild-id = discord服务ID
      - mj.discord.channel-id = discord服务组ID
      - mj.queue.timeout-minutes = 6
      - mj.api-secret = abc123456
      - mj.discord.user-token = **********


```
