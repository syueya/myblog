## compose部署

```
services:
  ollama:
    image: ollama/ollama:latest
    container_name: ollama
    restart: unless-stopped
    ports:
      - 11434:11434
    volumes:
      - ./data:/root/.ollama # 模型存储目录
    environment:
      - OLLAMA_HOST=0.0.0.0:11434
```


## 翻译

官网：https://ollama.com/huihui_ai/hy-mt1.5-abliterated

使用浑源翻译模型 `1.5：huihui_ai/hy-mt1.5-abliterated:1.8b`
- 容器好了以后进入终端
- 然后输入命令：`ollama run huihui_ai/hy-mt1.5-abliterated:1.8b`
- 运行成功以后，就可以使用ollama翻译了
