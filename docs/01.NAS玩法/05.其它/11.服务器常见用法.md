## 安装软件包与依赖

先使用`sudo apt update`更新软件包索引。


### 安装Docker官方版

```
# 安装4个基础依赖
apt install -y ca-certificates curl gnupg lsb-release

# ========== 添加密钥 ==========
# 创建密钥目录，权限 755
install -m 0755 -d /etc/apt/keyrings
# 下载 Docker 官方 GPG 密钥
curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
# 给密钥文件添加读取权限 
chmod a+r /etc/apt/keyrings/docker.asc


# ========== 添加软件源（国内用阿里云）==========
# 添加软件源（国内用阿里云）
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://mirrors.aliyun.com/docker-ce/linux/ubuntu jammy stable" | tee /etc/apt/sources.list.d/docker.list > /dev/null
# 添加软件源后更新软件包索引
apt update


#  安装 Docker 引擎和 Compose 插件
apt install -y docker-ce docker-ce-cli containerd.io docker-compose-plugin


# 启动 Docker 服务 
systemctl start docker
# 设置开机自启
systemctl enable docker

# 验证 Docker 是否安装成功
docker --version
docker compose version
```

验证后发现我们已经安装了 Docker 和 Docker Compose：
- Docker版本：Docker version 29.3.0, build 5927d80
- Docker Compose版本：Docker Compose version v5.1.0

如果我们想在运行中的容器内执行命令可以使用命令：`docker exec [选项] 容器名/ID 命令 [参数]`
- 选项: -i 输入交互模式，-t分配伪终端

比如连接pgsql：`docker exec -it pgsql psql -U postgres`


查看容器状态

### 安装nvm

1、安装node管理器nvm

```bash
# 下载并执行安装脚本
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# 或国内加速：
curl -o- https://gitee.com/mirrors/nvm/raw/v0.39.7/install.sh | bash


# 使 NVM 生效
source ~/.bashrc

# 验证 nvm 的安装
nvm --version
```

2、安装node

```
# 查看可用版本
nvm ls-remote | grep "v22"

# 安装 Node 22 最新版
nvm install 22

# 或指定具体版本
nvm install 22.11.0

# 设置默认版本
nvm use 22
nvm alias default 22

# 安装 pnpm
npm install -g pnpm

# 验证 node 和 npm 的安装
node -v   # v22.x.x
npm -v    # 10.x.x
pnpm -v   # 9.x.x
```

常用 NVM 命令：

| 命令 | 作用 |
|:---|:---|
| `nvm install 22` | 安装 Node 22 |
| `nvm use 22` | 切换到 Node 22 |
| `nvm ls` | 查看已安装版本 |
| `nvm ls-remote` | 查看远程可用版本 |
| `nvm alias default 22` | 设置默认版本 |
| `nvm uninstall 20` | 卸载指定版本 |








### 写入文件

```
cat > 文件名 << '结束符'
内容...
结束符
```

结束符有：1. EOF 2. END 3. STOP 4. 任意字符串
```
# 用 EOF
cat > file.txt << 'EOF'
hello world
EOF

# 用 END
cat > file.txt << 'END'
hello world
END

# 用 STOP
cat > file.txt << 'STOP'
hello world
STOP

# 用任意字符串
cat > file.txt << 'ABC123'
hello world
ABC123

```

比如创建 docker-compose.yml 文件：
```
cat > docker-compose.yml << 'EOF'
services:
  redis:
    image: redis:latest
    container_name: redis
    restart: always
    ports:
        - 6379:6379
    volumes:
        - ./data:/data
    command: redis-server --appendonly yes --requirepass yourpassword
EOF
```
验证文件内容：`cat docker-compose.yml`
