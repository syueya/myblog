## ssh连接方法
可以使用系统自带的终端用命令行连接，也可以用专门的客户端来连接。

### 命令行连接

语法：`ssh [options] [user@]hostname [command]`

参数说明：
- `[user@]hostname`
  - user : 要连接的远程主机的用户名
  - hostname  远程主机的名称或IP地址。
- `[command]`：可选的在远程主机上执行的命令。

常用选项：
- `-l user`：指定要登录的用户。
- `-p port`：指定连接到远程主机的端口号，默认是22。
- `-i identity_file`：指定身份验证文件（私钥文件）。
- `-v`：详细模式，可以显示调试信息。
- `-C`：启用压缩。
- `-N`：不执行远程命令，只进行端口转发。
- `-f`：后台运行。
- `-L local_port:remote_host:remote_port`：本地端口转发。
- `-R remote_port:local_host:local_port`：远程端口转发。
- `-D [bind_address:]port`：动态应用程序级端口转发。


示例：
```
# 如果你的用户名是xia，服务器地址是192.168.31.15
ssh xia@192.168.31.15

# 如果指定了ssh端口为222
ssh -p 222 xia@192.168.31.15
```
