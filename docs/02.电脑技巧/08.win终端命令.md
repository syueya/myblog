## 命令添加 PATH 环境变量

比如把 `D:\tools\uv` 添加到 PATH 环境变量中：
```
# 把 D:\tools\uv 加入用户 PATH（User）
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";D:\tools\uv", [EnvironmentVariableTarget]::User)

# 把 D:\tools\uv 加入系统 PATH（Machine）
[Environment]::SetEnvironmentVariable("Path", [Environment]::GetEnvironmentVariable("Path","Machine") + ";D:\tools\uv", "Machine")
```

说明:
- SetEnvironmentVariable(变量名, 值, 作用范围)
- Windows 环境变量有三个作用范围：
  - Process：当前终端进程
  - User：当前用户
  - Machine：整个系统（所有用户）
- 上述命令等价于在 Windows 图形界面中操作：系统 → 高级系统设置 → 环境变量 → Path → 新增 D:\tools\uv

注意事项：
- 修改 Machine（系统变量） 需要使用 管理员权限 打开 PowerShell。
- 修改完成后，需要 重新打开终端，新的 PATH 才会生效。
